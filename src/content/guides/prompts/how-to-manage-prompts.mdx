## Introduction

Manage your prompts more efficiently by learning how to add, star, set defaults, and delete prompts you no longer need.  
This guide walks you through each feature step by step.

## Frequently Asked Questions

**Q: Can I favorite more than one prompt?**  
A: Yes, you can star as many prompts as you like. It helps organize your favorites for quick access.

**Q: Is there only one default prompt?**  
A: Yes, only one prompt can be set as the default at a time.

**Q: What happens when I delete a prompt?**  
A: Deleted prompts are permanently removed and cannot be recovered.
