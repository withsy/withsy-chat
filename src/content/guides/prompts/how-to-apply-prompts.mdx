## Prompt Application Guide

Prompts let you control how the assistant responds during chats. Follow this guide to apply a prompt to your current conversation or manage your default prompt.



### View the Currently Applied Prompt  
The right sidebar in a chat shows the currently **applied prompt**. If no prompt is active, the assistant will behave according to system defaults or your set default.

<img
  src="/guides/default-and-apply-prompt.png"
  alt="Currently applied prompt in chat"
  className="my-6 rounded-lg shadow-sm w-full max-w-2xl mx-auto"
/>

Click **Apply** next to any prompt to apply it to the current chat.



### Apply or Clear a Prompt  
To apply a prompt, find it in the list and click the **Apply** button.

> Just clicking the card does not apply the prompt. You must click **Apply**.

To remove it, click **Clear** in the sidebar.

<img
  src="/guides/default-and-clear-prompt.png"
  alt="Clear or apply prompt"
  className="my-6 rounded-lg shadow-sm w-full max-w-2xl mx-auto"
/>



### Star or Unstar Prompts  
On the [Prompts page](/prompts), use the **⋮ menu** on a prompt card to manage it:

- **Star** to favorite it  
- **Unstar** to remove it from favorites  
- **Make Default** to apply it automatically to all new chats  
- **Delete** to remove it permanently

<img
  src="/guides/prompts-dropdown.png"
  alt="Prompt options: star, make default, delete"
  className="my-6 rounded-lg shadow-sm w-full max-w-xl mx-auto"
/>



### Set or Change Your Default Prompt  
You can **only change the default prompt on the Prompts page**, not from the sidebar.
<br />
From the **⋮ menu**:

- Select **Make Default** to apply it automatically to new chats  
- Or choose **Remove Default** to unset it

<img
  src="/guides/default-dropdown.png"
  alt="Default prompt menu options"
  className="my-6 rounded-lg shadow-sm w-full max-w-xl mx-auto"
/>
